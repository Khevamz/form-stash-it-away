Create forms and store submissions easily with Form Stash plugin.

== Description ==

Form Stash is a powerful yet easy-to-use form builder plugin that allows you to create custom forms and collect submissions from your website visitors.

Features:

* Intuitive drag-and-drop form builder
* Multiple field types: text, email, textarea, dropdown, checkbox
* Form validation
* Submission storage in your WordPress database
* Export submissions to CSV
* Shortcode to embed forms on any post or page
* Responsive design

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/form-stash` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the "Form Stash" menu in your admin sidebar to create and manage forms
4. Use the shortcode [form_stash id="X"] to display your form on any post or page (replace X with your form ID)

== Frequently Asked Questions ==

= How do I create a new form? =

Navigate to the Form Stash page in your WordPress admin, enter a form name, add fields as needed, and click "Save Form".

= How do I add a form to my page? =

Use the shortcode [form_stash id="X"] where X is the ID of your form. You can find the ID in the Form Builder interface.
